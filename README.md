# RAG-Based QA Chatbot

An AI-powered Retrieval-Augmented Generation (RAG) chatbot that provides accurate and context-aware responses using institutional document.

# Features
- Semantic search using vector embeddings
- Context-aware response generation
- Reduced hallucination
- Fast and accurate information retrieval

## Tech Stack

- **Backend:** Python
- **Frameworks:** FastAPI, LangChain
- **Frontend:** Streamlit
- **Vector Database:** Qdrant
- **Containerization:** Docker (for Qdrant)
- **Embedding Model:** Nomic Embed Text
- **LLM:** Ollama (Phi3 Mini)
